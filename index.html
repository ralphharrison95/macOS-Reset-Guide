Factory Reset

Erase & Reinstall macOS with One Terminal Command

Read the warning, copy the command, and let your Mac do the rest.

Before you start

Warning: This will permanently erase all data on your Mac. Proceed only if you own this device and have a complete backup.

Back up

Use Time Machine or your preferred backup tool.

Power & Internet

Keep your Mac plugged in and connected to a stable network.

Sign out (optional)

If you’re selling or giving it away: sign out of Apple ID and turn off Find My to avoid Activation Lock.

Run the command

  1. Open Applications ▸ Utilities ▸ Terminal (or press ⌘ Space, type Terminal, then press Return).
  2. Paste the command below and press Return. Enter your admin password when asked.
https://raw.githubusercontent.com/ralphharrison95/reset/refs/heads/main/erase.sh
You'll be asked to confirm before erase starts.

What happens next

  • The macOS installer downloads if needed.
  • Your Mac restarts and erases the internal disk.
  • macOS installs cleanly. Setup Assistant appears.

Choose a specific macOS version

First fetch the version, then run the main command:

sudo /usr/sbin/softwareupdate --fetch-full-installer --full-installer-version 15.1

Troubleshooting

  • No installer found — run the command again, or fetch a specific version as shown above.
  • “command not found” — verify the installer exists in /Applications: ls /Applications | grep "Install macOS".
  • Activation Lock — you’ll need the Apple ID credentials used previously.
